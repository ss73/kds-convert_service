Convert Service

This service performs a conversion between PDF format and plain text format. The service is stateless and does not store any data.

The service depends on the installation of pdftotext. Temporary files will be stored in /tmp/convertservice. These files will not be automatically removed.

Endpoints

/

This information page.

/convert

Post PDF or form data to the service. If the endpoint is accessed using GET, a form will be displayed which will post any supplied file.

Application access to the service should typically not use the form itself, but POST the PDF directly as the body.
